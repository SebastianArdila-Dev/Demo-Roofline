# Roofline — Demo Paula

Sitio web de demostración premium para **Roofline Company**, diseñado con estética Apple-inspired: dark mode, tipografía SF Pro, animaciones fluidas con GSAP + Lenis, efectos parallax y CTA directo a WhatsApp.

---

## Estructura del proyecto

```
Demo Paula/
├── index.html              ← Página principal (self-contained)
├── README.md               ← Este archivo
├── DESIGN.md               ← Sistema de diseño y tokens
└── assets/
    ├── css/
    │   └── main.css        ← Estilos (también inline en index.html)
    ├── js/
    │   └── main.js         ← JavaScript (también inline en index.html)
    └── images/             ← (vacío — visuals generados con CSS)
```

## Cómo abrir

Doble clic en `index.html` — se abre directo en el navegador. No requiere servidor.

## Tecnologías usadas

| Tecnología | Uso |
|---|---|
| HTML5 / CSS3 | Estructura y estilos |
| GSAP 3.12 + ScrollTrigger | Animaciones de scroll |
| Lenis 1.0 | Smooth scrolling |
| CSS Custom Properties | Design tokens |
| CSS Grid + Flexbox | Layouts responsivos |
| Inter / SF Pro (-apple-system) | Tipografía |

## Secciones del sitio

1. **Nav** — Navegación flotante con efecto blur al scroll
2. **Hero** — Pantalla completa con orbes animados y texto reveal
3. **Marquee** — Categorías de productos en scroll infinito
4. **Servicios** — 3 tarjetas con efecto 3D tilt
5. **Showcase** — Bloques alternados: Vinilos / Maquinaria / Tintas
6. **Stats** — Contadores animados al entrar en viewport
7. **Por qué Roofline** — 3 propuestas de valor
8. **CTA** — Llamado a acción directo a WhatsApp
9. **Footer** — Links y datos de contacto
10. **WhatsApp Float** — Botón flotante con pulse animation
